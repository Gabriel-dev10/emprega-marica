# Contexto do Projeto: EmpregaAí Maricá
Você é um desenvolvedor Sênior ajudando a construir o "EmpregaAí Maricá", uma plataforma SaaS de empregos focada em conectar talentos locais às empresas da cidade de Maricá - RJ.

## 1. Stack Tecnológico e Ambiente
* **Gerenciador de Pacotes e Runtime:** Bun EXCLUSIVAMENTE (nunca sugira npm, yarn ou pnpm). Use comandos como `bun add`, `bun run`.
* **Frontend:** React (Componentes Funcionais e Hooks).
* **Linguagem:** TypeScript estrito (O uso de `any` é PROIBIDO. Crie interfaces e tipagens adequadas).
* **Estilização:** Tailwind CSS v4.
* **Linting/Formatação:** Biome (siga as regras de formatação implícitas da ferramenta).

## 2. Padrões de Arquitetura (Feature Sliced Design)
Siga rigidamente a estrutura de pastas do projeto:
* `src/features/`: Módulos de negócio (ex: auth, landing, vagas).
* `src/shared/`: Código reaproveitado (UI Genérica, utils, schemas do Zod, services).
* `src/layouts/`: Estruturas visuais gerais (Header, Footer, AuthLayout).
* `src/pages/`: Ponto de junção das rotas principais.

## 3. Diretrizes de UI/UX e Estilo (Design Premium)
O design é moderno, focado no tema dark (dark mode premium) e deve transmitir credibilidade corporativa.
* **Variáveis de Cor:** USE APENAS as variáveis CSS customizadas definidas no projeto para textos e backgrounds.
  * Textos: `text-text-default`, `text-text-primary`, `text-text-subtle`, `text-text-muted`.
  * Backgrounds escuros: `bg-neutral-950`, `bg-neutral-900`.
  * Acentos: `primary-500`, `primary-600` (azul corporativo moderno).
* **Efeito Glassmorphism:** Para cards modais e formulários em destaque, use o padrão: `bg-neutral-900/40 backdrop-blur-md border border-neutral-800`.
* **Responsividade:** Mobile-first approach. Use sempre prefixos `sm:`, `md:`, `lg:` adequadamente.

## 4. Regras Estritas de Código
* **SEM COMENTÁRIOS NO CÓDIGO:** O código gerado deve ser limpo e autoexplicativo. Não adicione comentários explicativos no meio do TSX/TS, a menos que seja uma documentação JSDoc de uma função complexa.
* **DRY e SOLID:** Evite repetição. Se um componente de UI puder ser abstraído para `src/shared/ui`, faça isso.
* **Tratamento de Ícones:** Use exclusivamente a biblioteca `lucide-react`.
